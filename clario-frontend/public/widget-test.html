How It Works under the hood
  1. Pasting the widget token initializes the iframe viewport dynamically targeting http://localhost:5173/widget/:token.
  2. The parent wrapper communicates theme changes and title text via the postMessage API protocol.
  3. Subsequent visitor conversations and SSE streamings call raw backend routers on port 3000.
